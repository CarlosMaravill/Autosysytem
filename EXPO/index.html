¿QUE ES AUTOSYSTEM?

El proyecto tiene como finalidad ser la herramienta de confianza del área enderezado y pintura, ya que al tratarse de un área laboral con falta de opciones o posibilidades administrativas muchas veces datos, registros o demás información se pierde. La aplicación de nuestro sistema permitirá el sencillo y completo manejo de datos en función laboral de talleres de enderezado y pintura, tomando control sobre todo dato recibido de las posibles entradas a registro. Se cuenta con las herramientas necesarias para que el programa cumpla con las tareas solicitadas por parte del cliente, además de ser una opción mucho mas eficiente que el papeleo común, también contara con la seguridad necesaria para así garantizar el total control de estos datos sin riesgo de perdidas o alteraciones.

REQUERIMIENTOS



Especificaciones mínimas

Estas son las especificaciones mínimas y necesarias para el sistema AUTO SYSTEM:

SO*: Windows 10 64-bit (v.1709 o superior)
CPU: Intel Core i3-4340 o AMD FX-6300
RAM: 4 GB de memoria RAM
Disco duro: 15 GB de espacio libre
DirectX: requiere un sistema compatible con DirectX 11
Red: conexión a Internet de banda ancha

Especificaciones recomendadas

Estas son las especificaciones recomendadas para que el sistema funcione correctamente sin ningun tipo de fallas tecnicas:

SO: Windows 10 64-bit con su última actualización
CPU: Intel Core i5-2500K
RAM: 8 GB de memoria RAM
Disco duro: 17 GB de espacio libre
Red: conexión a Internet de banda ancha
